在线词典助手-升级说明

您已升级至在线词典助手0.4版本。如需完整文档或插件最新信息，请访问 项目主页

0.5版新增内容
  1. 在快速选项处增加了切换牌组选项，以便为不同词典选择不同牌组
  2. 更新柯林斯英汉/英英词典脚本，使例句中单词加粗，以便anitmoon制卡
  3. 为后台脚本提供了任意音频播放功能。现已支持朗文6等MDX词典原声例句播放
0.4版新增内容
  1. 极大的提高了查询速度，大幅度优化弹窗响应时间。
  2. 新增词典脚本：法语助手、西语助手，千亿俄汉，汉典辞典。
  3. 修复bug，完善文档说明，在选项页增加脚本清单链接

本插件本身不含任何词典数据，仅以脚本获取在线词典信息，下面是部分脚本支持的词典名称，完整内容详见脚本清单

  1. 简明词典：有道简明英汉，百词斩图文词典
  2. 权威词典：牛津双解，朗文双解，剑桥双解，柯林斯双解
  3. 其他词典：法语助手、西语助手、千亿俄汉、汉典汉语
  4. 其他双语：柯林斯英法/法英、剑桥英法/法英词典

提示：ankiconnect官版不支持同一单词重复制卡。想重复制卡，详见这篇博客文章

Online Dictionary Helper -- Upgrade Notes

You extension was upgraded to v0.4. Here is the release note. If you need more details, please visit Project Home

version 0.5
  1. Added deckname selection in quick option.
  2. Updated internal Collins script to support specified Anki 'antimoon' template
  3. Support 'PlaySound' function for any given embeded audio on webpage.
version 0.4
  1. Significantly improved response time and performance.
  2. Added new script: 'frcn_Eudict, escn_Eudict, rucn_Qianyi, cncn_Zdic'.
  3. Fixed bug, update document and add script full list in option page.

The extension doesn't contain any dictionary data. It only uses script to retrive data from online dictioanry. Below is part of scripts list. For detail, please check Full List

  1. Concise：encn_Youdao, encn_Baicizhan
  2. Learner：encn_Oxford, encn_Longman, encn_Cambridge, encn_Collins
  3. Others：frcn_Eudict, escn_Eudict, rucn_Qianyi, cncn_Zdic
  4. Bilingual: Collins(enfr/fren), Cambridge(enfr/fren)
